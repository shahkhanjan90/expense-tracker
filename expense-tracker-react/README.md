# Expense Tracker React

A React and Vite version of the expense tracker application.

## Features

- Dashboard for expense tracking and summaries
- Monthly configuration and target management
- Category and recurring expense management
- Charts and utilization views
- Local persistence through the app services layer

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```
